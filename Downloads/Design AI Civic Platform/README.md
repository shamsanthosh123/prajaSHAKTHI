# 📌 Problem Statement

**Community Hero – Hyperlocal Problem Solver**

Communities frequently face civic issues such as potholes, water leakages, damaged streetlights, garbage accumulation, road damage, and public infrastructure problems. Traditional reporting systems are fragmented, lack transparency, and make issue tracking difficult.

**PrajaShakthi** bridges this gap by providing an AI-powered platform where citizens can instantly report issues, receive AI-generated analysis, track complaint status, and interact with an intelligent civic assistant.

# ✨ Features

## 📸 AI Image Analysis

- Upload civic issue images
- AI automatically identifies issue category
- Predicts severity level
- Suggests responsible government department
- Generates complaint title and description
- Supports English & Kannada

---

## 🤖 AI Civic Assistant

- Powered by Google Gemini
- Answers citizen queries
- Explains complaint status
- Guides users through civic services
- Uses complaint history as context

---

## 📍 Smart Location Detection

- Reverse Geocoding
- Displays exact complaint location
- Location attached to every report

```bash

```


Deploy

